Smart Solid Waste Pollution Prevention
System (SSWPPS)

Description

The Smart Solid Waste Pollution Prevention System (SSWPPS) is an IoT project aimed at providing a solution for garbage disposal in drains and rivers. It is equipped with sensors that provide real-time data on the disposal level and is connected to a user-friendly management system, allowing for effective management and monitoring of any drains that are integrated with the system. The first picture shows the actual product which is pure civil engineering without the element of software engineering that has been implemented in Australia, while the second picture is a prototype that I have created and innovate major systematic feature implemented on the project.

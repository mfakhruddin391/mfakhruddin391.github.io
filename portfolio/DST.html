Dropship Stock Tracker (DST)

Description

Dropship Stock Tracker (Keeptrackit) System is a product management system designed to facilitate communication between dropship agents and distributors. The system provides specific information about stock quantities to help streamline the process for both parties. The idea was developed in response to the challenges faced by dropship merchants, as a result of a thorough study. The main issue for dropship merchants and distributors is the time and effort required to answer repeated inquiries about stock quantities from numerous stockist agents. This system aims to make it easier for them to quickly and accurately respond to stockist inquiries about product availability.

Technology

  • PHP
  • MySQL
  • Bootstrap/CSS

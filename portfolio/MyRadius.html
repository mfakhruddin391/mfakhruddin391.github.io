MyRadius

Description

MyRadius is a mobile application that contributes to the mitigation of COVID-19 spread by implementing a virtual boundary. During the Movement Control Order (MCO) phase, MyRadius features a functionality for monitoring individual movements. One of the government's Standard Operating Procedures (SOPs) for COVID-19 dictates that individuals are only permitted to leave their residence within a 10-kilometer radius. MyRadius features a function that effectively discourages individuals from exceeding the radius. Upon scanning the MySejahtera QR code prior to entering a store or grocery outside of their city, the user will receive a notification indicating their location within or outside of the specified radius.

Technology

  • Flutter
  • Dart
  • Firebase
  • Google Cloud Platform (Map API)
